# agenda-em-c
A simple contact management system built in C.  
This project allows you to add, remove, list, and persist contacts using dynamic memory allocation and binary file storage.

---

## Features

- Add new contacts
- Remove existing contacts
- List all contacts
- Save contacts to a file
- Load contacts automatically on startup

---
## Concepts Used

- Structs
- Pointers
- Dynamic memory allocation (`malloc`, `realloc`, `free`)
- File handling (`fopen`, `fwrite`, `fread`)
- CLI (Command Line Interface)
